# Code Assembly (拼好码)

> 能复用时不重造，能编排时不发明。

为 Codex 等 AI 编程 Agent 设计的方法论 skill。每次编码任务开始时加载。

## 核心流程

1. **写清需求** — 目标、输入、输出、约束、验收标准
2. **反向搜索** — ①官方能力 → ②社区成熟工具 → ③平台服务
3. **评估候选** — 维护状态、许可证、文档、生产案例、生态兼容
4. **选择组合** — 记录为什么选、为什么不选、为什么不自研
5. **设计边界** — 接口契约、错误处理、依赖隔离、回滚路径
6. **生成胶水** — 只写连接/适配/编排/配置/业务规则，不重写能力本身
7. **验证交付** — 类型检查、集成测试、验收逐条过

## 原则

- 搜索不可跳过
- 评估不可省略
- 记录不可偷懒
- 回滚路径必须有
- 验证才等于完成
