# code-assembly-skill

**拼好码** — AI 编程的整合者心态。

能复用时不重造，能编排时不发明。Search-first approach: reverse-search mature toolchains before writing any code.

## 核心理念

每次新任务，必须先按以下流程执行，不走完不得动手写代码：

1. **写清需求** — 目标、输入、输出、约束、验收标准
2. **反向搜索** — 拆能力 → ①官方 CLI/Skills/MCP → ②社区成熟工具 → ③平台服务
3. **评估候选** — 维护状态、stars、许可证、文档、生产案例
4. **选组合+决策记录** — 为什么选、为什么不选、为什么不自研
5. **设计边界** — 适配层、回滚路径
6. **只写胶水代码** — 不重写成熟能力内部逻辑
7. **验证交付** — 逐条过验收标准

## 适用场景

- 任何技术任务开始前的调研阶段
- agent 频繁造轮子时
- 需要选择技术方案时
